A performant, block-first WordPress theme supporting editorial blogging, portfolio showcases, and guided onboarding.

== Description ==

Aetheria is a modern, performance-focused WordPress theme built with the block editor in mind. It features:

* Full Site Editing (FSE) support
* 13 custom blocks for enhanced functionality
* Portfolio custom post type
* Testimonial custom post type
* 3 style variations (Serif Editorial, Monochrome, Soft Dark)
* Dark mode support
* Onboarding wizard
* Performance optimizations
* Accessibility compliance (WCAG 2.1 AA)
* WooCommerce integration
* Multilingual and RTL ready

== Installation ==

1. Upload the theme files to /wp-content/themes/aetheria/
2. Activate the theme through the 'Appearance > Themes' screen
3. Follow the onboarding wizard to configure your site

== Frequently Asked Questions ==

= Does this theme support full site editing? =

Yes, Aetheria is built as a block theme with full FSE support.

= Is it compatible with WooCommerce? =

Yes, basic WooCommerce integration is included.
